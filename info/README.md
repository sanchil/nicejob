## Installation

```bash

npx express-generator --git --no-view nicejob
cd nicejob
npm i --save @google-cloud/firestore
npm i --save dotenv nanoid
npm i --save-dev @babel/cli @babel/core @babel/node @babel/preset-env 
npm i --save-dev nodemon rimraf
npm i --save-dev jest

```
